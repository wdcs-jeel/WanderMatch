## Installation

1. Clone the repository:

```bash
git clone https://github.com/yourusername/wandermatch.git

```
2. Install dependencies:
```
npm install --f
```
3. Clean and Rebuild (optional if need):
```
cd android
./gradlew clean

cd ios
xcodebuild clean
```
##  Running the App

Android
```
npx react-native run-android
```
iOS
```
cd ios
pod install
